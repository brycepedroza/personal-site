The Azure 500 course teaches students how to implement best security practices in the cloud. A certified Azure Security Engineer is one who can successfully stand-up common security controls and ensure that their organization has a strong security posture.

Working as an information security engineer at Starbucks, it's critical that application development teams do everything in their power to ensure that they are adhering to best cloud security practices. By completing this course, I hope to not only know how to implement security controls, but to also enable a frictionless security experience for engineers and Starbucks customers.

**These are the notes that I have taken in preparation for the exam. Hopefully you find them as useful as I have.**

The exam tests candidates on four key areas.
